symbol Concept

A powerful tool.

Grep (Globel Search Regular Expression and Printing out the line) is a powerful text search tool that is a search job for rows that can use regular expressions to search for text , And print the matching lines.

Grammar:

grep options ‘pattern’ file
command options search pattern file will be searched


Options Concept
-r Recursive search
-v [Reverse selection, only rows that do not match the pattern]
-o Only display the strings that are matched by the pattern, not the entire row
-i [Match is not case-sensitive
-A Show matching rows, the way behind the # line (# represents the value)
-B Preceding # lines
-C Before and after the # line
-E Use extended regular expressions
