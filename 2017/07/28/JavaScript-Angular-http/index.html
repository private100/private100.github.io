$http service

The $http service is a core AngularJS service that facilitates communication with the remote HTTP servers via the browser’s XMLHttpRequest object or via JSONP.

The $http service is a function to generate an HTTP request and returns a promise. And the promise tells user whether the request is accepted or rejected.
