Some notes about URL exception processing

URLError

The cause of URLError is usually caused by: no network connection or the target server does not exist. In this case, the exception object will have the reason attribute (a tuple of the (error code, cause of the error)).

HTTPError

A HTTP response returned from the server has a status code. Among them, some status code that the server can not complete the corresponding request, the default handler can handle some of status codes.

The default handler for the urllib2 module can handle redirection (status code is 300 range), and the status code is successful in the range of 100-299. Therefore, the range of status codes that can cause the HTTPError exception is: 400-599.
