When Item is collected in the Spider, it will be passed to the Item Pipeline, and some components will process the Item in a certain order.

The following are some typical applications of the item pipeline:

  • Clean up HTML data
  • Verify crawled data (check item contains some fields)
  • Check (and discard)
  • Save the crawled results, such as saved to the database, XML, JSON and other documents
